## 📖 Descripción

Este repositorio reúne los ejercicios, prácticas y modelos desarrollados durante el curso de **Investigación de Operaciones**, implementados en **Python** utilizando **GurobiPy** para formular y resolver problemas de optimización matemática.

Su propósito es servir como material de estudio, práctica y consulta para reforzar los conceptos de optimización y modelado matemático.

---

## 📚 Contenido

- 📈 Programación Lineal
- 🔢 Programación Entera
- ✅ Programación Binaria
- 🚚 Problemas de Transporte
- 👥 Problemas de Asignación
- 🌐 Modelos de Redes
- 🏭 Planeamiento de Producción
- 🎯 Optimización Multiobjetivo
- 📋 Casos de estudio y ejercicios de aplicación
